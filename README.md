### üìö Key Topics Breakdown

<details>
<summary><strong>Click to expand detailed topic list</strong></summary>

#### Part I: Abstract Algebra
*   **Set Theory:** Relations, Partitions, Equivalence classes.
*   **Groups:** $(G, \cdot)$, Abelian groups, Morphisms, Kernel and Image.
*   **Rings and Fields:** Properties of $(\mathbb{Z}, +, \cdot)$, Polynomial rings $K[X]$.

#### Part II: Linear Algebra
*   **Vector Spaces:** Vector spaces over a field $K$, Linear independence/dependence.
*   **Matrices & Determinants:** Matrix operations, Rank, Invertibility ($det(A) \neq 0$).
*   **Linear Maps:** Kernel ($\ker f$), Image ($\text{Im} f$), Rank-Nullity Theorem.
*   **Spectral Theory:** Characteristic polynomial $P_A(\lambda) = \det(A - \lambda I_n)$, Diagonalization.
*   **Bilinear Forms:** Symmetric forms, Positive definite forms, Signature.

#### Part III: Coding Theory
*   **Linear Codes:** Error detection and correction.
*   **Matrices:** Generator Matrix ($G$), Parity-Check Matrix ($H$).
*   **Decoding:** Syndrome decoding, Hamming distance.
